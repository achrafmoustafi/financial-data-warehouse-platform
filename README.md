# 🌍 Project Overview

This project focuses on the design and implementation of a **financial data warehouse** to support advanced analytics, business intelligence, and financial performance monitoring.

The solution is built using a **multi-layer data architecture (Bronze / Silver / Gold)** to ensure data quality, traceability, and scalable analytical reporting.

The project includes:

- SQL-based financial data warehouse design
- ETL/ELT transformations across Bronze, Silver, and Gold layers
- Star schema modeling with fact and dimension tables
- Analytical Power BI dashboards for financial reporting and KPI monitoring

This project is designed as a **senior-level analytics engineering and business intelligence case study** for financial data environments.

# 🏗️ Data Architecture

The warehouse follows a medallion-style architecture:

## Bronze Layer
Raw ingestion layer containing source financial transactions with minimal transformation.

## Silver Layer
Cleaned, standardized, validated, and enriched financial data prepared for business logic.

## Gold Layer
Business-ready analytical model with fact and dimension tables optimized for reporting and dashboards.
